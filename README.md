# GenAI
GitHub cannot render notebooks containing widget metadata generated by Google Colab.
The following nbconvert command was used to remove only the metadata while preserving all cell outputs, ensuring the notebook renders correctly on GitHub:
jupyter nbconvert --to notebook --ClearMetadataPreprocessor.enabled=True --inplace HandsOn-1_Unit1.ipynb
jupyter nbconvert --to notebook --ClearMetadataPreprocessor.enabled=True --inplace Unit1_Benchmark.ipynb
